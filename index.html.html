file upload :

The `input type="file"` tag in HTML is used to create a file upload control, allowing users to select and upload files from their local device. This tag is often used in forms to collect files such as images, documents, or videos. The `accept` attribute can be used to specify the types of files that the server accepts, for example, `accept=".jpg,.png,.pdf"`. Additionally, JavaScript can be used to enhance the file upload process, such as validating file types and sizes before submission. File uploads are commonly used in web applications for tasks like profile picture uploads, document submissions, and more.

Image tag :

The `img` tag in HTML is used to embed images in a webpage. It requires the `src` attribute, which specifies the path to the image file, and the `alt` attribute, which provides alternative text for the image if it cannot be displayed. The `width` and `height` attributes can be used to define the dimensions of the image. Additionally, the `title` attribute can be added to provide a tooltip when the user hovers over the image. The `img` tag is an inline element, meaning it does not start on a new line and only takes up as much width as necessary. Proper use of the `img` tag enhances the visual appeal and accessibility of a webpage.
